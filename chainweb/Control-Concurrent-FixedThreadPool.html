Runs an IO action asynchronously. Returns an Action you can wait on with waitAction.

runActionGroup :: Traversable t => ThreadPool -> t (IO ()) -> IO (Maybe SomeException) Source #

Runs a group of IO actions and waits for them to finish. If any of the IO actions throws an exception, runActionGroup will return it. If more than one action throws an exception, runActionGroup will arbitrarily return one of the exceptions. The runActionGroup function will wait until all of the actions have run, even if an exception is thrown.
