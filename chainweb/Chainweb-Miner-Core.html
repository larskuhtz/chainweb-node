fastCheckTarget :: Ptr Word64 -> Ptr Word64 -> IO Bool Source #

PowHashNat interprets POW hashes as unsigned 256 bit integral numbers in little endian encoding, hence we compare against the target from the end of the bytes first, then move toward the front 8 bytes at a time.

injectNonce :: Nonce -> Ptr Word8 -> IO () Source #

injectNonce makes low-level assumptions about the byte layout of a hashed BlockHeader. If that layout changes, this functions need to be updated. The assumption allows us to iterate on new nonces quickly.
